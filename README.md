# Shimpei Kobayashi MkDocs Site

This is a first MkDocs conversion draft based on the public Google Sites homepage.

## Local preview

Install MkDocs and Material:

```bash
pip install mkdocs mkdocs-material pymdown-extensions
```

Run locally:

```bash
mkdocs serve
```

Build static files:

```bash
mkdocs build
```

## Publish on GitHub Pages

1. Create a GitHub repository, e.g. `kobayashi-site`.
2. Upload all files in this folder.
3. Enable GitHub Pages, or run:

```bash
mkdocs gh-deploy
```
